    Library

    In the second semester, I developed a Library app, which advanced beyond the simple To-Do list. This app allows for detailed modification of list elements and incorporates more variables for added functionality. Additionally, the list entries are stored in localStorage, ensuring that the data persists even after the page is refreshed, providing a more dynamic and user-friendly experience.
