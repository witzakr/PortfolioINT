    Pokemon API

    This website is a demonstration of my work with a basic API. I used the Pokémon API to create a simple display (Pokédex) that showcases my ability to work with APIs. The site displays each Pokémon along with their evolutions, and when you click on one, it provides detailed stats. The design is kept simple, as the focus of this assignment was primarily on API usability rather than building a full-fledged application.
